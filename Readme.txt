
!Project Files Overview!

This repository contains the following key files essential for the functionality and interface of the project:

 1. sunflower.ino
This file contains the Arduino code responsible for controlling the operation of the sunflower project.

 2. templates.zip
This archive includes the interface design for the chess game and the implementation of the scoring system.

 3. Cam2.py
This file comprises the complete program code, managing all aspects of the system, including player moves, robot arm movements, and the capture of the chessboard state.

Each file plays a critical role in ensuring the seamless operation of the project components.
